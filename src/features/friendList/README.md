## 완료된 것
- 페이지의 프레임 및 섹션 정의
- 친구 추가 다이얼로그 정의
- 친구 목록, 친구 요청 카드 생성
- 전용 라벨 컴포넌트 생성
- 더미 데이터 등록
- css 구조 및 기본 디자인 적용

## 버그 기록
- 렌더링을 friendListItems에서 전역 컨텍스트인 friendData로 변경하는 과정에서 버그 발생. 원인은 컴포넌트 내부에서 직접 setFriendData(useState)를 사용하여 friendData를 변경하고, 또 변경된 컴포넌트를 렌더링하고, 또 변경하는 무한 루프에 빠짐.
