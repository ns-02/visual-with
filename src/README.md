# src/

앱의 모든 작동하는 코드가 포함된 폴더  
`App.tsx`와 `index.tsx`는 앱의 시작점 역할

## 주요 파일 및 폴더
- `App.tsx` : 라우팅 (예정), 전역 레이아웃 연결
- `index.tsx` : React를 브라우저에 연결하는 진입점
- `features/` : 실제 기능별 코드
- `components/` : 공용 UI
- `layouts/` : 레이아웃 구조
- `pages/` : 라우팅되는 페이지 (예정)
- `utils/` : 재사용 가능한 함수들
