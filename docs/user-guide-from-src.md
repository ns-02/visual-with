# Visual With 기능 안내 (사용자용)

이 문서는 `src` 하위의 기능별 README 내용을 사용자 관점으로 묶어 정리한 안내서입니다.
개발 구조 설명은 줄이고, 실제로 무엇을 할 수 있는지 중심으로 정리했습니다.

## 서비스에서 할 수 있는 일

- 로그인/회원가입 후 협업 공간에 접속
- 팀을 만들고, 팀을 전환하며 작업
- 팀원과 팀 채팅으로 소통
- 친구를 추가하고 1:1 채팅으로 대화
- 일정, 할 일, 파일을 한 화면 흐름 안에서 관리

## 주요 기능 현황

### 계정/접근
- 로그인, 회원가입 화면과 기본 접근 제어가 연결되어 있습니다.
- 로그아웃 동작은 대화상자 방식으로 제공됩니다.

### 팀 관리
- 팀 목록 확인, 팀 생성/삭제, 팀 전환 기능이 준비되어 있습니다.
- 팀 초대 흐름(링크/ID 기반)은 일부 시나리오가 계속 보완 중입니다.

### 팀 채팅
- 채팅 목록 확인과 메시지 전송(버튼/엔터)이 가능합니다.
- 파일 업로드, 채팅 검색은 기본 구조가 있고 고도화가 진행 중입니다.

### 친구/1:1 채팅
- 친구 목록/요청 목록 확인, 요청 수락/거절, 친구 삭제 흐름이 있습니다.
- 친구 선택 후 1:1 채팅 전송이 가능하며, 일부 검색 기능은 보완 중입니다.

### 파일 공유
- 파일 목록, 업로드, 삭제 흐름이 준비되어 있습니다.
- 드래그 앤 드롭 업로드는 동작 구현이 완료되었습니다.
- 다운로드/검색은 기본 단계로, 추가 개선이 예정되어 있습니다.

### 일정 관리
- 월/일 단위 일정 조회, 일정 추가/수정/삭제가 가능한 구조입니다.
- 연도 단위 탐색 등 일부 확장 기능은 추후 추가 예정입니다.

### 할 일 관리
- 할 일 목록 확인, 추가/수정/삭제, 완료 체크가 가능한 구조입니다.
- 검색 기능은 기본 단계로 유지 중입니다.

## 이용 시 참고

- 기능별 화면은 같은 레이아웃 규칙을 사용해 이동 흐름이 일관되도록 구성되어 있습니다.
- 일부 기능은 실제 서버 연동 기준으로 계속 개선 중이므로, 화면 동작과 세부 결과가 단계적으로 업데이트될 수 있습니다.
