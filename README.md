# Visual With

다기능 협업 툴 Visual With의 프론트엔드 <br>
React와 TypeScript로 개발 중 <br>
사용자 친화적인 인터페이스 목표

## 시작 방법
```
npm install
npm run dev
```
http://localhost:5173

## 프로젝트 구조
- [`src/` : 앱의 실제 코드](src/README.md)

## 기술 스택 및 도구
- React + TypeScript: 컴포넌트 추상화, 타입 안정성
- Context API: 전역 상태 관리
- CSS Modules: 단위 스타일 충돌 방지
- Storybook: 재사용되는 컴포넌트 관리 및 UI 일관성 확보
- Vite: 빌드 도구
- lucide-react: 아이콘 불러오기
- radix-ui: ui 제작 도구
- React Router: 리액트 환경에서 라우팅 제공
- React DayPicker: 날짜 선택 ui 지원
- ESLint: 정적 코드 분석 도구
- Prettier: 코드 포맷터
